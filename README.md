# prehack-bellmen
# Recruitment & Career Development Application
# Scope
This project is a website that makes finding and hiring for jobs easy. Companies can post jobs and see people who apply. Job seekers can search for jobs and apply quickly using their profile and resume. It is a simple tool to help companies hire the right people and help users find the proper jobs.
# Key Features
1) Company posting => Companies can create job posts to recruit people and view applicants who applied to their posts. 
2) Company acceptance => Companies can either accept or reject applicants’ requests.
3) Search and filter by keywords/jobs => Job seekers can search and filter for jobs through posts created by companies.
4) Easy Application => Job seekers can submit their applications immediately using their profiles and resumes for a more streamlined process.
5) User/Company Profile => Both the company and the user have separate profiles, and they are editable.
6) Company editing/deleting post => Companies can either edit or delete their posts.
7) User/Company Profile Edit => Both companies and users can edit their profiles to better represent themselves/their companies.
